# Agent Communication Protocols

This directory contains the communication standards and message schemas for agent-to-agent collaboration.

## Overview

The protocol defines how agents discover, handshake, communicate, and coordinate tasks.

## Protocol Layers

1. **Discovery** - How agents find each other
2. **Handshake** - How agents establish communication
3. **Messaging** - How agents exchange information
4. **Task Coordination** - How agents delegate and collaborate on tasks

## Version

Current Protocol Version: **1.0.0**
